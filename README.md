**Digital Tools Marketplace**

## 📖 Description

This is a modern web application where users can explore different digital tools, choose pricing plans, and manage their cart easily. The UI is clean, responsive, and user-friendly, inspired by real-world product platforms.



## 🛠️ Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🌼 DaisyUI
* 🔥 React Toastify
* 📦 JavaScript (ES6)



## ✨ Features

* 🛒 Add to Cart functionality with dynamic updates
* 💳 Pricing plan selection with interactive UI
* 👀 Conditional rendering (hide/show sections like Cart, Pricing, Started)
